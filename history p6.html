API HISTORY

Que es API HISTORY

El objeto DOM window proporciona acceso al historial del navegador a través del objeto history . Este da acceso a métodos y propiedades útiles que permiten avanzar y retroceder a través del historial del usuario, así como --a partir de HTML5-- manipular el contenido del historial.


Viajando a través del historial

Retroceder y avanzar a través del historial del usuario utilizando los métodos back(), forward() y go().


importancia

La importancia de este comando seria para un uso de facilitacion para la navegacion en las paginas, y el uso sobre nuestra pagina seria como regresar al INDEX o o regesar donde estabamos una vez ya estando ahi, un simple uso del boton regresar y volver


back() Method

Esta regresa a la pagina anterior, si no existe no lo hara

forward() Method

Esta volvera a la pagina, si no hubo una pagina abierta previamente no lo hara

go(-2) Method

Esta retrocedera en el historial hasta la antepenultima pagina abierta, si no hay mas no lo hara
